LAGAKU - Platform Turnamen Mahasiswa

LAGAKU adalah platform turnamen mahasiswa yang memudahkan pengelolaan, partisipasi, dan pelacakan kompetisi dengan teknologi modern.

🚀 Fitur Utama
📅 Halaman Utama (Dashboard)
Statistik Real-time: Menampilkan jumlah turnamen, pertandingan, tim aktif, dan jadwal hari ini

Filter Status Pertandingan: Berlangsung, Dijadwalkan, Selesai, Semua

Hero Section Modern: Dengan efek gradient dan call-to-action yang menarik

Responsif: Tampilan optimal untuk desktop dan mobile

📋 Halaman Jadwal Pertandingan
Tabel Jadwal Interaktif: Menampilkan seluruh jadwal pertandingan dengan filter lengkap

Filter Multi-Kriteria:

Pencarian berdasarkan tim atau turnamen

Filter berdasarkan tanggal (Hari Ini, Besok, Minggu Ini)

Filter berdasarkan status (Berlangsung, Dijadwalkan)

Filter berdasarkan turnamen tertentu

Detail Pertandingan: Modal dialog untuk melihat informasi lengkap pertandingan

Statistik Cards: Visualisasi data pertandingan aktif

👥 Manajemen Pengguna
Dual Role System: Admin dan Mahasiswa

Profil Lengkap: NIM, fakultas, program studi, data pribadi

Verifikasi Akun: Sistem verifikasi untuk keamanan

🏆 Sistem Turnamen
Acara/Turnamen: Pembuatan dan pengelolaan turnamen

Sistem Gugur: Bracket tournament dengan round-based matches

Tim & Anggota: Pendaftaran tim dan manajemen anggota

Babak (Round): Konfigurasi babak dengan jumlah tim minimum/maksimum

⚽ Manajemen Pertandingan
Status Multi-level: Dijadwalkan, Berlangsung, Selesai

Detail Lengkap: Skor, lokasi, waktu, durasi

Google Maps Integration: URL lokasi untuk navigasi

Pemenang Otomatis: Sistem penentuan pemenang berdasarkan skor

🛠️ Teknologi yang Digunakan
Frontend
Next.js 14 dengan App Router

TypeScript untuk type safety

Tailwind CSS untuk styling

shadcn/ui untuk komponen UI

Lucide React untuk ikon

Supabase Client untuk real-time database

Backend & Database
Supabase (PostgreSQL)

Authentication & Security dengan Row Level Security

Real-time subscriptions

Storage untuk file upload
